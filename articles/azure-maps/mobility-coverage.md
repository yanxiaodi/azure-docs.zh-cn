# <a name="azure-maps-mobility-coverage"></a>Azure Maps 移动性覆盖率

Azure Maps[移动服务](https://aka.ms/AzureMapsMobilityService)允许公共传输、 自行车共享和路由与 multimodal 摩托车共享的行程规划选项的功能。 公共传输，该服务提供邻近的测量站，搜索将停止，和行并提供信息，如计划扫描和实时公共传输到达的请求，停止和路由指南，以及详细的多模分步路线的列表和服务将发出警报。 移动服务也会返回实时和到汽车的静态可用性信息共享车辆和共享的自行车和踏板。

Azure Maps 不具有相同级别的信息和所有城市和国家/地区的准确性。 调用传输数据的能力取决于所在的城市，数据可能未包括的所有公共传输选项和提供市/县的机构。

下表提供了有关 Azure Maps 移动服务覆盖区域信息。


| 符号 | 含义 |
|--------|---------|
| *      |国家/地区也几乎完全被涵盖。|

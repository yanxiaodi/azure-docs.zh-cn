## <a name="run-code"></a>运行代码

使用 Node.js 在命令行中运行该应用程序。

调用 `npm start` 将添加话语、进行训练并获取训练状态。

```console
> npm start 
```

此命令行显示调用“添加陈述”API 的结果。

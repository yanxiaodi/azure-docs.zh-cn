# <a name="use-the-storsimple-device-manager-service-to-troubleshoot-the-storsimple-virtual-array"></a>使用 StorSimple Device Manager 服务排查 StorSimple 虚拟阵列问题
## <a name="overview"></a>概述

StorSimple Device Manager 在服务摘要边栏选项卡中提供了“诊断并解决问题”  设置，其中突出显示了虚拟阵列可能会出现的一些常见问题以及如何解决这些问题。 本教程介绍了 StorSimple Device Manager 服务中提供的自助服务故障排除功能。

![诊断并解决问题](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-main.png)

## <a name="diagnose-and-solve-issues"></a>诊断并解决问题

可以直接在 StorSimple Device Manager 服务摘要边栏选项卡中查看 StorSimple 虚拟阵列的一些常见问题，并查看这些问题的解决方案。

#### <a name="to-diagnose-an-issue-with-your-virtual-array"></a>诊断虚拟阵列问题

1. 在 StorSimple Device Manager 服务摘要边栏选项卡的左窗格中单击“诊断并解决问题”  设置，可查看常见问题列表。

2.  展开你遇到的问题的症状，以查看推荐步骤  ，帮助你解决问题。 如果需要，还可以查看提供的详细文档链接供将来参考。
   
    ![解决问题](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-offline.png)

3. 如果找不到问题的参考或无法解决问题，请联系 Microsoft 支持人员以获得进一步帮助。

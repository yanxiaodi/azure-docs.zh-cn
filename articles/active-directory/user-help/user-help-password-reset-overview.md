# <a name="reset-your-own-work-or-school-password-overview"></a>概述：重置自己的工作或学校密码

如果忘记了工作或学校密码、从来没有从组织处得到过密码，或者帐户被锁定，则可使用安全信息和移动设备来重置工作或学校密码。

你的管理员必须启用此功能，否则你无法注册信息和重置你自己的密码。 如果看不到“忘记密码”选项，则表明管理员尚未为组织启用此功能。  如果你认为这是错误，请联系支持人员以获取帮助。

>[!Important]
>此内容适用于用户。 如果你是管理员，可以在 [Azure Active Directory 文档](https://docs.microsoft.com/azure/active-directory)中查找有关如何设置和管理 Azure Active Directory (Azure AD) 环境的详细信息。

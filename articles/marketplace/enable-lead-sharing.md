# <a name="enable-lead-sharing"></a>启用潜在顾客共享
确保在 [Azure 市场](https://azuremarketplace.microsoft.com)或 [Microsoft AppSource](https://appsource.microsoft.com) 中为你的列表启用了潜在顾客管理，可以帮助你获取以下受众的联系信息：

| 店面 | 目标受众 |
|:--- |:--- |
| AppSource | 业务客户 |
| Azure 市场 | 所有客户 |

潜在顾客为下表中列出的活动、现场销售人员的相关销售活动以及应用或产品/服务性能情况的相关信息提供了拓展基础：

| 店面 | 活动 |
|:--- |:--- |
| AppSource | 巩固关系 |
| Azure 市场 | 按需生成 |

针对潜在顾客使用以下做法：
*   评估潜在顾客并根据销售机会为其评分。
*   巩固与潜在客户的关系，以使潜在客户进入销售管道。
*   根据整体市场营销活动策略开展工作。

潜在客户展现了客户对以下列表项的兴趣：

| 店面 | 商品 |
|:--- |:--- |
| AppSource | <table> <tr><td>应用</td></tr> <tr><td>咨询服务</td></tr> </table> |
| Azure 市场 | <table> <tr><td>产品/服务</td></tr> <tr><td>技术</td></tr> </table> |

你应该设置一个流程来跟进潜在顾客，如下表所述：

| 店面 | 潜在顾客 |
|:--- |:--- |
| AppSource | <table> <tr><td>你的 CRM</td></tr> <tr><td>Microsoft 现场销售人员 CRM</td></tr> </table> |
| Azure 市场 | <table> <tr><td>Azure 市场</td></tr> <tr><td>Microsoft 现场销售人员 CRM</td></tr> </table> |

市场店面中的潜在顾客不同于 Microsoft 销售人员限定的潜在顾客（联合销售计划的一项功能）。

若要详细了解如何访问联合销售计划，请参阅[通过 Microsoft 促进业务发展](./promote-your-business-with-microsoft.md)。

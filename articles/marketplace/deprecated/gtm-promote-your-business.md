# <a name="promote-your-business-with-microsoft"></a>通过 Microsoft 促进业务发展

市场合作伙伴需要加入 Microsoft 合作伙伴网络 (MPN)。 作为 MPN 成员，你可以获取以业务为中心的其他权益。 请访问 [partner.microsoft.com](https://partner.microsoft.com)，以了解如何利用该网络获得以下好处：

* 新的业务机会。
* 与团队或合作伙伴联系。
* 有助于培养技能组合的解决方案和培训。

## <a name="benefits-and-resources"></a>权益和资源

若要推广产品/服务或应用，以及获取更多权益和资源，请使用下表：

| 优点 | 资源 |
|:--- |:--- |
| 利用核心权益 | 作为 Microsoft 合作伙伴网络的成员，你会收到一系列核心权益，既省时又省钱。 权益可以帮助你 <ul> <li> 增强能力， </li> <li> 更好地服务客户，并 </li> <li> 建立业务关系，从而实现你的全部业务潜能。 </li> </ul> 有关核心权益的详细信息，请访问[核心权益和要求](https://partner.microsoft.com/membership/core-benefits#tab-content-1)。 |
| 赢得云平台资格 | 获取资格有助于 <ul> <li> 让你从其他 Microsoft 合作伙伴中脱颖而出。 </li> <li> 证明拥有技术专长，并可帮助客户在市场上取得成功。 </li> </ul> 资格是加入许多关键合作伙伴计划（如联合销售）的先决条件。 有关详细信息，请参阅[云平台资格](https://partner.microsoft.com/reach-customers/selling-with-microsoft)。 |
| 做好 IP 联合销售准备 | 你可能有资格加入 IP 联合销售就绪计划。  借助此计划，可以实现以下目标： <ul> <li> 就目标客户机会和客户规划，与 Microsoft 销售人员和其他合作伙伴直接协作。 </li> <li> 让你的解决方案可供 Microsoft 销售人员在销售人员解决方案目录中查看。 </li> <li> Microsoft 销售人员会竭诚与你协作，开创双赢局面。 </li> </ul> 有关详细信息，包括资格要求，请参阅[联合销售就绪计划和促进业务发展](https://partner.microsoft.com/reach-customers/selling-with-microsoft)。 |

# <a name="consulting-service-prerequisites"></a>咨询服务先决条件

请在[Azure Marketplace 和 AppSource 的咨询服务](../../consulting-services.md)中查看此产品/服务的说明和要求。  以下特定咨询服务类型具有其他要求：

|     产品/服务类型   |    要求   |
|     ----------   |    -----------   |
| Microsoft Azure | 合作伙伴必须为[云平台资格](https://partner.microsoft.com/membership/cloud-platform-competency)认证提供银或金牌认证。 |
| Dynamics 365 for Customer Engagement | 合作伙伴必须具有针对[云客户关系管理能力](https://partner.microsoft.com/membership/cloud-customer-relationship-management-competency)的银牌或金牌级认证。 |
| Dynamics 365 for Finance & Operations | 合作伙伴必须拥有[企业资源计划资格](https://partner.microsoft.com/membership/enterprise-resource-planning-competency)的银牌或金牌级认证。 合作伙伴在过去12个月内的云运营中还必须具有最小收入 $25000。 |
| Dynamics 365 Business Central | 合作伙伴必须已经是至少一个客户的云解决方案提供商 (CSP) 或数字记录合作伙伴 (DPOR)。 <br/> 有关如何成为 CSP 的信息，请参阅[从作为 CSP 开始](https://partner.microsoft.com/cloud-solution-provider/get-started)。 <br/> 有关 DPOR 要求的信息，请参阅 [DPOR 概述](https://partner.microsoft.com/membership/digital-partner-of-record#Membership_Digital_Partner_of_Record_Navigated_Rich_Text_Node1)。 |
|  Power BI  | 在 Power BI 合作伙伴展示上列出。 有关详细信息，请参阅[解决方案合作伙伴条件](https://powerbi.microsoft.com/become-a-partner)。 |
|  PowerApps       | 合作伙伴必须具有在[合作伙伴解决方案展示](https://powerapps.microsoft.com/partner-showcase)上发布的解决方案。 |
|  |  |


## <a name="next-steps"></a>后续步骤

满足上述要求后，就可以[创建咨询服务了](./cpp-consulting-service-create-offer.md)。

# <a name="threat-modeling-tool-update-release-71510231---1112018"></a>Threat Modeling Tool 更新版 7.1.51023.1 - 2018 年 11 月 1 日

正如最初在[正式版说明](threat-modeling-tool-releases-71509112.md)中所提到的，我们发布了 Microsoft Threat Modeling Tool 的更新 (7.1.51023.1)，该更新将要求预览版（版本 < 7.1.50911.2 的预览客户端）用户升级到支持的正式版。 此版本不包含任何新功能或修补程序。

- 预览版的用户在打开客户端时将自动下载升级。 如果你选择不安装新更新，则该工具的预览版将关闭。
- 该工具的正式版将提示用户选择是否要升级。
- 该工具的 2016 版的用户将不受影响。

## <a name="feature-changes"></a>功能更改

- None

## <a name="system-requirements"></a>系统要求

- 支持的操作系统
  - Microsoft Windows 10
- 所需的 .NET 版本
  - .NET 3.5.2
- 其他要求
  - 需要建立 Internet 连接才能接收工具和模板的更新。

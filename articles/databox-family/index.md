<h1>Azure Data Box</h1>
<p>Azure Data Box 系列提供不同存储容量的产品，可帮助以快速、廉价且可靠的方式将数 TB 的数据发送到 Azure。 Microsoft 提供支持脱机或网络数据传输的专有存储设备，从而加快安全数据传输速度。</p>

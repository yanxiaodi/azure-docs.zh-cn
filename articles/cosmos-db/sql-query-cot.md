# <a name="cot-azure-cosmos-db"></a>COT （Azure Cosmos DB）
 返回指定数值表达式中指定角度的三角余切。  
  
## <a name="syntax"></a>语法
  
```sql
COT(<numeric_expr>)  
```  
  
## <a name="arguments"></a>参数
  
*numeric_expr*  
   为数值表达式。  
  
## <a name="return-types"></a>返回类型
  
  返回数值表达式。  
  
## <a name="examples"></a>示例
  
  下面的示例计算指定角度的 @no__t 0。  
  
```sql
SELECT COT(124.1332) AS cot  
```  
  
 结果集如下。  
  
```json
[{"cot": -0.040311998371148884}]  
```

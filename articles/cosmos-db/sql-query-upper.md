# <a name="upper-azure-cosmos-db"></a>上部（Azure Cosmos DB）
 返回在将小写字符数据转换为大写后的字符串表达式。  
  
## <a name="syntax"></a>语法
  
```sql
UPPER(<str_expr>)  
```  
  
## <a name="arguments"></a>参数
  
*str_expr*  
   字符串表达式。  
  
## <a name="return-types"></a>返回类型
  
  返回字符串表达式。  
  
## <a name="examples"></a>示例
  
  下面的示例演示如何在查询中使用 @no__t 0  
  
```sql
SELECT UPPER("Abc") AS upper  
```  
  
 结果集如下。  
  
```json
[{"upper": "ABC"}]  
```

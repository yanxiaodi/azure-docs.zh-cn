# <a name="azure-resource-manager-templates-for-app-service"></a>应用服务的 Azure 资源管理器模板

若要了解应用服务资源的 JSON 语法和属性，请参阅 [Microsoft.Web 资源类型](/azure/templates/microsoft.web/allversions)。

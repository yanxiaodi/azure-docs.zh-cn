# <a name="azure-functions-documentation"></a>Azure Functions 文档

Azure Functions 是一个无服务器计算服务，使用它可以按需运行代码，而无需显式预配或管理基础结构。 使用 Azure Functions 可以运行脚本或代码片段，以响应各种事件。 参考我们的快速入门、教程和示例了解如何使用 Azure Functions。

## <a name="clean-up-resources"></a>清理资源

在前面的步骤中，在资源组中创建了 Azure 资源。 如果将来不再需要这些资源，可以通过删除资源组来删除它们。
 
在 Azure 门户的左侧菜单中，选择“资源组”  ，然后选择“myResourceGroup”  。

在资源组页中，确保列出的资源是要删除的资源。

选择“删除”  ，在文本框中键入 **myResourceGroup**，然后选择“删除”  。

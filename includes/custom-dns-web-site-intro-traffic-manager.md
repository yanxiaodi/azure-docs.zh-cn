使用 Microsoft Azure 流量管理器对传往 Azure 网站的流量执行负载均衡时，可以使用 Azure 分配的 **\*.trafficmanager.net** 域名访问该网站。 您还可以关联自定义域名，例如 www\.contoso.com，与您的网站提供更易识别的域名，并为你的用户。

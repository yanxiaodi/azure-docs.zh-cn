## <a name="issue-custom-image-provisioning-errors"></a>问题：自定义映像;预配错误
上传或捕获用作专用 VM 映像的通用化 VM 映像时，将发生预配错误，反之亦然。 前者会导致预配超时错误，后者会导致预配失败。 若要部署自定义映像且不出错，必须确保在捕获过程中映像类型不会更改。

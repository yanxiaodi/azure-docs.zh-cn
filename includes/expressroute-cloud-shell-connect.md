 如果使用 Azure Cloud Shell，则到 Azure 帐户自动登录后单击试用。 若要本地登录，请使用提升的权限打开 PowerShell 控制台并运行 cmdlet 来连接。

```azurepowershell
Connect-AzAccount
```

如果有多个订阅，请获取 Azure 订阅的列表。

```azurepowershell-interactive
Get-AzSubscription
```

指定要使用的订阅。

```azurepowershell-interactive
Select-AzSubscription -SubscriptionName "Name of subscription"
```

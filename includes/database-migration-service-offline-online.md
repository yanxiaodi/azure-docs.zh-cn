> [!TIP]
> 使用 Azure 数据库迁移服务将数据库迁移到 Azure 时，可以进行脱机或联机迁移。   使用脱机迁移时，应用程序停机时间从迁移开始时算起。 使用联机迁移时，停机时间仅限在迁移结束时进行转换的那段时间。 建议对脱机迁移进行测试，以便确定其停机时间是否可以接受；如果不能接受，请进行联机迁移。

用于添加示例话语的 `response` 数组指示每个具有 `hasError` 属性的示例话语是成功还是失败。 以下 JSON 响应显示两个话语都已成功添加。 

```json
    "response": [
        {
            "value": {
                "UtteranceText": "go to seattle today",
                "ExampleId": -5123383
            },
            "hasError": false
        },
        {
            "value": {
                "UtteranceText": "book a flight",
                "ExampleId": -169157
            },
            "hasError": false
        }
    ]
```

以下 JSON 显示了训练请求成功的结果：

```json
{
    "request": null,
    "response": {
        "statusId": 9,
        "status": "Queued"
    }
}
```

1. 选择“应用程序”。 
1. 选择“webapi1”  应用程序以打开其属性页。 
1. 选择“已发布的范围”  。 可以使用已发布的范围向客户端应用程序授予访问 Web API 的某些权限。
1. 在“范围”中输入 `demo.read`，在“说明”中输入 `Read access to the web API`。  
1. 在“范围”中输入 `demo.write`，在“说明”中输入 `Write access to the web API`。  
1. 选择“保存”。

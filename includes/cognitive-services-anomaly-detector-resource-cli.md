Azure 认知服务由你订阅的 Azure 资源表示。 使用 [Azure 门户](../articles/cognitive-services/cognitive-services-apis-create-account.md)或 [Azure CLI](../articles/cognitive-services/cognitive-services-apis-create-account-cli.md) 在本地计算机上创建用于异常检测器的资源。 还可以：

* 免费获取在七天内有效的[试用密钥](https://azure.microsoft.com/try/cognitive-services/#decision)。 注册之后，它将在 [Azure 网站](https://azure.microsoft.com/try/cognitive-services/my-apis/)上提供。  
* 在 [Azure 门户](https://portal.azure.com/)上查看资源

## <a name="setupdevenv"></a>设置开发环境
首先，请通过 Visual Studio 和 Azure SDK 来设置开发环境。

* Visual Studio 2019 包含 Azure SDK。 如果使用的 Visual Studio 2019，无需其他设置则需要适用于开发环境。
* 对于 Visual Studio 2015，请单击以下链接，安装[用于 Visual Studio 2015 的 Azure SDK](https://go.microsoft.com/fwlink/?linkid=518003)。
* 对于 Visual Studio 2013，请单击以下链接，安装[用于 Visual Studio 2013 的 Azure SDK](https://go.microsoft.com/fwlink/?LinkID=324322)。

* 如果您尚未安装 Visual Studio，请使用以下命令安装[Visual Studio 2019](https://www.visualstudio.com/)与 Azure SDK。

> [!NOTE]
> 根据计算机上已有 SDK 依赖项的数量，安装 SDK 可能耗时较长，从几分钟到半小时或更长时间不等。
